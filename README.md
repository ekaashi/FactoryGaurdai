# 🏭 FactoryGuard AI

### 🚀 Predictive Maintenance System using Machine Learning

FactoryGuard AI is an intelligent predictive maintenance system that analyzes machine sensor data and predicts the likelihood of machine failure. It helps industries reduce downtime, optimize maintenance schedules, and improve operational efficiency.

---

## 📌 Overview

Traditional maintenance systems react **after failure occurs**, leading to downtime and increased costs.

**FactoryGuard AI solves this by:**

* Predicting failures **before they happen**
* Providing **real-time risk analysis**
* Enabling **data-driven maintenance decisions**

---

## ⚙️ Features

* 📊 Real-time failure prediction
* 🎯 Machine failure probability scoring
* 🟢🟡🔴 Risk level classification (Low / Medium / High)
* 📈 Interactive dashboard with charts
* 🧠 XGBoost-based ML model
* 🎛️ User-friendly Streamlit interface

---

## 🧠 Machine Learning Model

* **Algorithm:** XGBoost Classifier
* **Task:** Binary Classification (Failure / No Failure)
* **Output:** Probability score (0–1)

---

## 📥 Input Parameters

### Sensor Data:

* Temperature
* Pressure
